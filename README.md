## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/IgorRoc/igorroc.github.io

# Access
$ cd igorroc.github.io

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
